# PREVEN

방역 및 위생 관리 전문 기업 PREVEN의 반응형 기업 홈페이지 프로젝트입니다.

신뢰감을 전달하는 디자인과 직관적인 정보 구조를 통해 서비스 소개, 작업 과정, 문의 전환을 효과적으로 제공하는 것을 목표로 제작되었습니다.
